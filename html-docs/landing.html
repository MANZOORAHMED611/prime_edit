PrimeEdit is a production-grade text editor built in C++17 on Qt 6, designed for fast, reliable editing on Linux. It handles everything from quick config edits to multi-gigabyte log files, with the editing power of VS Code and the speed of a native application.

Built for real work

Three things that set PrimeEdit apart from every other Linux editor.

1GB

Large files, no compromise

Memory-mapped I/O with viewport loading. Only visible lines stay in memory. A 93MB minified JSON opens instantly at ~1MB RSS.

  • Files over 1GB at <200MB RAM
  • Minified file detection
  • Chunked lazy loading
  • Background search with progressive results
II

Full editing power

Everything you expect from a modern editor, without the Electron overhead.

  • Multi-cursor editing (Ctrl+D, Ctrl+Shift+L)
  • Split-screen views
  • 96 syntax languages
  • LSP integration (go-to-def, rename, diagnostics)
  • Git gutter indicators and commit dialog
  • Plugin system with C++ API
  • Remote file editing via SSH
C++

Native performance

Not Electron. Not a wrapper. A ground-up C++ application that starts in under a second and stays fast.

  • ~7MB binary
  • <1 second startup
  • PieceTable buffer with unlimited undo
  • 9 built-in themes
  • Session restore and crash recovery
  • Desktop integration (file associations, icons)
