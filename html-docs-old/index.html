Project Overview

Olive Notepad is a native Linux text editor designed to match and exceed Notepad++ functionality while incorporating modern features from VS Code and Sublime Text. Built with Qt6 and C++17, it provides a lightweight, fast, and deeply integrated experience with the Linux desktop environment.

Key Features

✓ Core Editing
  • Tabbed interface with drag-drop reordering
  • Multi-encoding support (UTF-8, UTF-16, etc.)
  • Line ending detection (LF, CRLF, CR)
  • Unlimited undo/redo history
  • Syntax highlighting (13+ languages)
✓ Search & Replace
  • Find and replace with regex support
  • Case-sensitive and whole-word matching
  • Replace all with preview
  • Go to line navigation
  • Incremental search highlighting
✓ Line Operations
  • Duplicate, delete, move lines
  • Sort lines (ascending/descending)
  • Remove duplicate lines
  • Join lines with space
  • Trim trailing whitespace
  • Toggle line comments
✓ Text Transformations
  • UPPERCASE conversion
  • lowercase conversion
  • Title Case conversion
  • Smart auto-indentation
  • Tab/space conversion
⚙ Session Management
  • Auto-restore open files on startup
  • Crash recovery system
  • Recent files tracking
  • Persistent window geometry
  • Auto-save to recovery location
🎨 Customization
  • Configurable font and size
  • Tab width settings (2, 4, 8 spaces)
  • Word wrap toggle
  • Line numbers display
  • Current line highlighting
  • Zoom in/out/reset

Quick Start

Building from Source

# Install dependencies (Ubuntu/Debian)
sudo apt install build-essential cmake qt6-base-dev qt6-base-dev-tools

# Clone repository
cd notepad_supreme

# Create build directory
mkdir build && cd build

# Configure and build
cmake ..
make -j$(nproc)

# Run the application
./Notepad\ Supreme
