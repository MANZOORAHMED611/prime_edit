1. Line Operations

Comprehensive set of line manipulation commands for efficient text editing.

Implementation Details:

  • Files Modified: src/ui/editor.h, src/ui/editor.cpp, src/ui/mainwindow.cpp, src/ui/mainwindow.h
  • New Methods: 12 new public methods in Editor class
  • Menu Integration: "Line Operations" submenu under Edit menu

Features:

Operation Shortcut Description
Duplicate Line Ctrl+Shift+D Duplicates current line or selection below
Delete Line Ctrl+Shift+K Deletes entire line including newline
Move Line Up Alt+Up Swaps current line with line above
Move Line Down Alt+Down Swaps current line with line below
Sort Lines (Ascending) - Sorts selected lines alphabetically A-Z
Sort Lines (Descending) - Sorts selected lines alphabetically Z-A
Remove Duplicate Lines - Removes duplicate lines from selection
Join Lines - Joins selected lines with spaces
Trim Trailing Whitespace - Removes trailing spaces from all lines

Technical Implementation:

void Editor::duplicateLine()
{
    QTextCursor cursor = textCursor();
    cursor.beginEditBlock();  // Atomic undo operation

    // Select entire line(s)
    cursor.movePosition(QTextCursor::StartOfBlock);
    cursor.movePosition(QTextCursor::EndOfBlock, QTextCursor::KeepAnchor);

    QString text = cursor.selectedText();
    cursor.movePosition(QTextCursor::EndOfBlock);
    cursor.insertText("\\n" + text);

    cursor.endEditBlock();
}
2. Case Conversion

Text transformation utilities for changing case of selected text.

Features:

  • UPPERCASE: Converts selection to all capitals
  • lowercase: Converts selection to all lowercase
  • Title Case: Capitalizes first letter of each word

Implementation:

  • Uses Qt's QString::toUpper() and QString::toLower()
  • Title case uses custom algorithm respecting word boundaries (space, dash, underscore)
  • Accessible via Edit → Convert Case menu
3. Toggle Comments

Language-aware comment toggling for rapid code commenting/uncommenting.

Supported Languages:

Language Comment Style
Python, Ruby, Bash, YAML # comment
C/C++, Java, JavaScript, Rust, Go // comment
SQL, Lua -- comment
HTML, XML <!-- comment -->

Behavior:

  • Detects if all selected lines are commented
  • If all commented: removes comments
  • If any uncommented: adds comments to all lines
  • Preserves indentation by adding comments after leading whitespace
  • Shortcut: Ctrl+/
4. File Explorer Panel

Side panel for browsing and opening files from the filesystem.

Features:

  • Tree view of directory structure
  • Filter files by name (incremental search)
  • Navigate up to parent directory
  • Double-click to open file or enter directory
  • Folder selection dialog
  • Refresh button to reload directory
  • Current path display with home directory abbreviation (~)

Implementation:

  • Base Class: QWidget with QVBoxLayout
  • Model: QFileSystemModel for efficient directory browsing
  • View: QTreeView with single-column mode
  • Signals:
    • fileDoubleClicked(QString) - emitted when file is opened
    • fileSelected(QString) - emitted when file is clicked
    • rootPathChanged(QString) - emitted when directory changes
